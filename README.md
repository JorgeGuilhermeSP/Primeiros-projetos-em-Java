# 🚀 Primeiros Projetos em Java

Bem-vindo ao meu repositório de portfólio de estudos em Java! Este espaço foi criado para centralizar e organizar os meus primeiros projetos e exercícios práticos na linguagem, demonstrando a minha evolução em lógica de programação, orientação a objetos e boas práticas de desenvolvimento.

---

## 🛠️ Tecnologias e Conceitos Utilizados

* **Linguagem:** Java (JDK)
* **IDE:** IntelliJ IDEA
* **Versionamento:** Git e GitHub (Configurado via SSH)
* **Conceitos Aplicados:**
    * Estruturas condicionais e de repetição (Lógica pura)
    * Orientação a Objetos (Classes, Atributos, Métodos, Pacotes)
    * Tratamento de Exceções Personalizadas (`CustomException`)
    * Padrões de Arquitetura de Dados simples (Camadas `model`, `dao` e `validator`)

---

## 📂 Estrutura do Repositório

O repositório está organizado em pastas independentes. Cada pasta contém um mini-projeto ou exercício com seu próprio código-fonte (`src`):

* **`📁 Cadastros`**: Um sistema de gerenciamento de cadastros mais robusto (desenvolvido durante os módulos da DIO), aplicando conceitos de persistência simulada com DAO (`UserDAO`), validações customizadas (`UserValidator`) e tratamento rigoroso de erros com exceções personalizadas.
* **`📁 IMC`**: Calculadora de Índice de Massa Corporal que recebe dados do usuário, processa os cálculos e devolve a classificação de saúde com base nas tabelas oficiais.
* **`📁 Tabuada`**: Exercício prático focado em estruturas de repetição (`loops`) para geração automática de tabelas matemáticas.
